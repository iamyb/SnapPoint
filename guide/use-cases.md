# Use Cases

SnapPoint adapts to different content types using context-aware agents, ensuring you get the most relevant summary structure for every source.

## Academic Papers (arXiv)
**Agent**: Paper Agent

<img src="/arxiv.png" alt="ArXiv paper summary preview" style="border-radius: 8px; margin: 1rem 0; border: 1px solid var(--vp-c-divider);">

Extract the core contributions, methodology, and limitations from dense academic writing without losing the essential structure.
- **Key Focus**: Contributions, experimental results, and future work.
- **Best For**: arXiv, research journals, and technical whitepapers.

## Online Discussions
**Agent**: Forum Agent

<img src="/hacknews.png" alt="Forum discussion summary preview" style="border-radius: 8px; margin: 1rem 0; border: 1px solid var(--vp-c-divider);">

Follow long threads on sites like Hacker News or Reddit. Capture the strongest arguments and surface the actual consensus instead of raw comment volume.
- **Key Focus**: Main arguments, controversial points, and community consensus.
- **Best For**: Hacker News, Reddit, and GitHub discussions.

## News & Articles
**Agent**: News Agent

<img src="/bbc.png" alt="News article summary preview" style="border-radius: 8px; margin: 1rem 0; border: 1px solid var(--vp-c-divider);">

Condense long-form reporting into a clean brief with core facts, context, and practical takeaways at a glance.
- **Key Focus**: Who, what, where, when, and why.
- **Best For**: BBC, New York Times, and independent blogs.

## GitHub Repositories
**Agent**: Repo Agent

<img src="/github.png" alt="GitHub repository summary preview" style="border-radius: 8px; margin: 1rem 0; border: 1px solid var(--vp-c-divider);">

Quickly understand the purpose and structure of a repository by summarizing READMEs, issue trends, and codebase overviews.
- **Key Focus**: Project goal, installation, and main features.
- **Best For**: GitHub repos and documentation landing pages.

## Finance & Market Updates
**Agent**: Finance & Stock Analysis

<img src="/yahoo.png" alt="Finance news summary preview" style="border-radius: 8px; margin: 1rem 0; border: 1px solid var(--vp-c-divider);">

Turn earnings stories, stock moves, and macro headlines into clear, investor-focused briefs.
- **Key Focus**: Market impact, key figures, and analyst sentiment.
- **Best For**: Yahoo Finance, Bloomberg, and market news.

## Technical Documentation
**Agent**: Technical Documentation

Quickly understand the purpose and structure of a project by summarizing its installation, core API, and main features.
- **Key Focus**: Project purpose, quick start, and core features.
- **Best For**: GitHub READMEs and official documentation.

## How-to & Tutorials
**Agent**: How-to & Tutorial

Summarize lengthy educational content into clear, actionable steps.
- **Key Focus**: Learning objectives, prerequisites, and core execution steps.
- **Best For**: Educational blogs, course modules, and technical guides.

## Product Reviews
**Agent**: Product Review & Comparison

Extract the essential pros, cons, and performance metrics from product reviews.
- **Key Focus**: Value proposition, strengths/weaknesses, and final verdict.
- **Best For**: Tech blogs and e-commerce reviews.

## Legal & Policies
**Agent**: Legal & Policy

Translate dense legal jargon into plain language.
- **Key Focus**: User obligations, data collection disclosure, and critical risks.
- **Best For**: Terms of Service and Privacy Policies.

---

## Custom Use Cases
SnapPoint allows you to define custom rules for specific domains. If none of the built-in agents fit, you can create your own instructions to handle unique content types.
